# Privacy Policy

EnvironmentNEPAL does not collect, store, or share any personal data.

- **No cookies** — We do not use tracking cookies.
- **No analytics** — We do not use Google Analytics or any third-party analytics.
- **No accounts** — There is no sign-up, login, or user accounts on this site.
- **No data sharing** — We have no data to share with third parties.

This site is hosted on GitHub Pages. GitHub may collect technical logs (IP addresses, request timestamps) as part of their standard hosting operations. See [GitHub's Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement) for details.

The AQI dashboard fetches data from IQAir/OpenWeather APIs using server-side stored keys. No user data is transmitted.
